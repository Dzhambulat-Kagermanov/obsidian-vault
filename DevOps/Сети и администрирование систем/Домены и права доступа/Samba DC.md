
### Основы:

**SAMBA** — это свободный программный пакет, который реализует протокол **SMB/CIFS** (Server Message Block / Common Internet File System) в Unix-подобных системах. Он позволяет Linux-серверам и рабочим станциям:

- Предоставлять файлы и принтеры для доступа из сетей Windows
- Интегрироваться в домены Active Directory
- Выступать в роли контроллера домена (начиная с версии 4.x)

#### Основные возможности:

|Функция|Описание|
|---|---|
|📁 Обмен файлами|Доступ к общим папкам из Windows, macOS, Linux|
|🖨️ Общий доступ к принтерам|Печать с любых устройств в сети|
|🔐 Аутентификация|Поддержка локальных пользователей, домена AD, LDAP|
|🌐 Интеграция с AD|Участие в домене Windows как клиент или контроллер|
|⚙️ Гибкая настройка|Детальный контроль прав доступа, шифрования, аудита|

#### Типичные сценарии использования:

1. Файловый сервер для смешанной сети;
2. Домашняя сеть: доступ к медиа с любого устройства:
	- Телевизор (DLNA через SAMBA);
	- Смартфон;
	- Ноутбук на Windows.
3. Корпоративная среда:
	- Интеграция Linux-сервера в Active Directory;
	- Централизованное управление пользователями и политиками;
	- Резервное копирование общих ресурсов.
4. Контроллер домена на базе Linux:
	- Domain Controller;
	- DNS-сервера;
	- Сервера политик групп (GPO — с ограничениями).
