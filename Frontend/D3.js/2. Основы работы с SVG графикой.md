
### Атрибут viewBox:

Атрибут `viewBox` имеет значение `0 0 900 300`. Первые два числа определяют начало системы координат viewBox (координаты x и y) - в данном случае обе эти координаты равны 0. Последние два числа - это ширина и высота видимой области. Они определяют соотношение сторон SVG и гарантируют, что он идеально масштабируется, чтобы вписаться в любой контейнер без искажений. То есть контейнер SVG теперь адаптируется к любому размеру экрана, сохраняя соотношение сторон 900:300.

### Текст:

Для отрисовки текста в SVG предназначен элемент `<text>`. При работе с этим элементом нужно использовать как открывающий, так и закрывающий теги, между которыми размещается собственно текст. Например:

```html
<svg viewBox="0 0 300 150" style`="border:1px solid black;">
    <text>Hello METANIT.COM</text>
</svg>
```

 Если мы загрузим страницу в браузере, то скорее всего мы ниего не увидим. Так как положение текста в SVG вычисляется относительно его базовой линии, контролируемойатрибутом `dominant-baseline`. Если координата базовой линии текста равна (0, 0), то фактический текст оказывается за пределами контейнера SVG. Поэтому мы не увидим текст на странице. Но мы можем использовать атрибуты x и y для размещения текста в определенной точке. Например:

```html
<svg viewBox="0 0 300 150" style`="border:1px solid black;">
    <text>Hello METANIT.COM</text>
</svg>
```

### Линии:

Элемент line представляет прямую линию между двумя точками. Для определения координат начальной и конечной точек этот элемент использует атрибуты x1 и y1 и x2 и y2 соответственно. Например:

```html
<svg>
  <line x1="10" y1="15" `x2="220" y2="200" stroke="black" stroke-width="2" />
</svg>
```

### Прямоугольники:

`<rect>` представляет прямоугольник. Для отрисовки элементу `<rect>` требуются четыре атрибута: x и y (положение верхнего левого угла прямоугольника), width и height (соответственно ширина и высота). Например:

```html
<svg>
  <rect x="15" y="10" width="120" height="60" />
</svg>
```

![[1.12.png|697]]

### Круг:

`<circle>` представляет окружность. Его обязательными атрибутами являются положение центра круга - атрибуты `cx` и `cy` и его радиус - атрибут `r`. Например, круг с центром в точке (x=100, y=80) и радиусом в 50 пикселей:

```html
<svg>
  <circle cx="100" cy="80" r="50" fill="#55efc4" stroke="#00b894" />
</svg>
```

![[1.17.png|697]]

### Эллипс:

`<ellipse>` представляет овал, требует атрибутов для положения центра фигуры - `cx` и `cy`. Но при этом также требует два атрибута-радиуса - горизонтальный радиус - атрибут `rx` и вертикальный радиус - атрибут `ry`. Например, овал с горизонтальным радиусом 50 пикселей и вертикальным радиусом 30 пикселей:

```html
<svg>
  <ellipse cx="150" cy="80" rx="100" ry="40" fill="#55efc4" stroke="#00b894" />
</svg>
```

### Геометрический путь:

Геометрический путь (контур) или элемент `<path>` в SVG представляет набор линий и широко используется в D3.js для рисования практически всех сложных форм и кривых, которые не могут быть представлены ни одной из ранее рассмотренных фигур. Его атрибут `d` содержит список команд, который указывают, где начать рисовать путь, где его завершить, какие типы кривых применять, должен ли быть контур замкнутым или нет. Простейший пример пути:

```html
<svg>
  <path d="M20 80 C 90 150, 105 150, 160 80 S 230 10, 300 80" fill="none" stroke="navy" stroke-width="3" />

</svg>
```
