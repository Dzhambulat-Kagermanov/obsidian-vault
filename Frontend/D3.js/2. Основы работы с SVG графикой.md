
### Атрибут viewBox:

Атрибут `viewBox` имеет значение `0 0 900 300`. Первые два числа определяют начало системы координат viewBox (координаты x и y) - в данном случае обе эти координаты равны 0. Последние два числа - это ширина и высота видимой области. Они определяют соотношение сторон SVG и гарантируют, что он идеально масштабируется, чтобы вписаться в любой контейнер без искажений. То есть контейнер SVG теперь адаптируется к любому размеру экрана, сохраняя соотношение сторон 900:300.

### Координатная система:

Система координат SVG начинается в верхнем левом углу контейнера SVG. Это означает, ось Y направлена сверху вниз.

### Текст:

Для отрисовки текста в SVG предназначен элемент `<text>`. При работе с этим элементом нужно использовать как открывающий, так и закрывающий теги, между которыми размещается собственно текст. Например:

```html
<svg viewBox="0 0 300 150" style`="border:1px solid black;">
    <text>Hello METANIT.COM</text>
</svg>
```

 Если мы загрузим страницу в браузере, то скорее всего мы ниего не увидим. Так как положение текста в SVG вычисляется относительно его базовой линии, контролируемойатрибутом `dominant-baseline`. Если координата базовой линии текста равна (0, 0), то фактический текст оказывается за пределами контейнера SVG. Поэтому мы не увидим текст на странице. Но мы можем использовать атрибуты x и y для размещения текста в определенной точке. Например:

```html
<svg viewBox="0 0 300 150" style`="border:1px solid black;">
    <text>Hello METANIT.COM</text>
</svg>
```

### Линии:

Элемент line представляет прямую линию между двумя точками. Для определения координат начальной и конечной точек этот элемент использует атрибуты x1 и y1 и x2 и y2 соответственно. Например:

```
`<``svg``>`

  <line x1="10" y1="15" `x2``=``"220"` `y2``=``"200"` `/>`

</svg>
```
