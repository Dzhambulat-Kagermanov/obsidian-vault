
С помощью привязки данных мы можем связать отдельный фрагмент данных с элементами DOM. Чтобы привязать данные, нужно использовать цепочку из трех методов (selectAll(), data() и join()). Метод data() получает через параметр данные и связывает их с элементыми, а метод join() добавляет html-элементы в DOM. Например:

```ts
const langs = ["JavaScript", "TypeScript", "Python", "Dart"];

d3.select("ul")           // выбираем элемент ul на веб-странице
    .selectAll("li")      // определяем элементы li для элемента ul
    .data(langs)          // связываем каждый элемент li с элементом массива langs
    .join("li")           // добавляем элементы li на веб-страницу
    .text(item=>item);    // устанавливаем текст для элементов li
```

В метод `text()` передается функция, параметр которой привязанный к html-элементу элемент массива (в нашем случае массива langs). Возвращаемое значение этой функции - тот текст, который надо вывести в html-элементе. То есть в качестве item здесь передается строка из массива langs и эта строка выводится в элементе:
