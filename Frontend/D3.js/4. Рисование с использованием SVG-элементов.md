### Создаем SVG:

Первоначально, надо создать SVG-элемент, в который мы будем включать все наши фигуры:
```ts
const w = 500
const h = 50

const svg = d3.select("body")
.append("svg")
.attr("width", w)
.attr("height", h)
```


### Фигуры на основе данных:

Создаем данные для отрисовки фигур:
```ts
const dataset = [ 5, 10, 15, 20, 25 ];
```

Создаем круги:
```ts
circles = svg.selectAll("circle")
.data(dataset)
.enter()
.append("circle");
```

Стилизуем круги:
```ts
circles
.attr("cx", function(d, i) {
	return (i * 50) + 25;
})
.attr("cy", h/2)
.attr("r", function(d) {
	return d;
});
```

### Создание столбчатой диаграммы:

```ts
const w = 500;
const h = 100;

const svg = d3.select("body")
.append("svg")
.attr("width", w)
.attr("height", h)

svg.selectAll("rect")
.data(dataset)
.enter()
.append("rect")
.attr("x", function(d, i) {
	return i * (w / dataset.length)
})
.attr("y", 0)
.attr("width", 20)
.attr("height", 100)
```

### Рисование путей:

Пути представляют более сложные объекты по сравнению с простыми примитивами. Путь позволяет отобразить набор любых примитивов с заполнением или без заполнения цветом. Для создания пути определяется элемент _path_:

```ts
const width = 400
const height = 400

const data = [
	{x: 80, y: 50},{x: 110, y: 80},{x: 140, y: 90},
	{x: 170, y: 70},{x: 200, y: 60},{x: 230, y: 60},
	{x: 260, y: 70},{x: 290, y: 80},{x: 320, y: 70}
];

// функция, создающая по массиву точек линии
const line = d3.svg.line()
	.x(function(d){return d.x;})
	.y(function(d){return d.y;});

const svg = d3.select("body").append("svg");

svg.attr("height", height).attr("width", width);

// добавляем путь
svg.append("path").attr("d", line(data));
```

![[3.6.png]]

Для определения содержимого пути определяется атрибут d, который может принимать следующие команды:

- M (абсолютные координаты) / m (относительные координаты): переместить на позицию (x y);
- Z / z: завершение пути;
- L / l: провести линию к точке (x y);
- C / c: провести кубическую кривую Безье к точке(x1 y1 x2 y2 x y);
- Q / q:  Провести кубическую кривую Безье к точке(x1 y1 x y);
- A / a: провести дугу по координатам (rx ry x-axis-rotation large-arc-flag sweep-flag x y).

И если мы посмотрим на сгенерированный код html, то увидим следующий элемент:

```html
<svg height="400" width="400">
	<path d="M80,50L110,80L140,90L170,70L200,60L230,60L260,70L290,80L320,70">
	</path>
</svg>
```
