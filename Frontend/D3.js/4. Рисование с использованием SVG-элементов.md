### Создаем SVG:

Первоначально, надо создать SVG-элемент, в который мы будем включать все наши фигуры:
```ts
const w = 500
const h = 50

const svg = d3.select("body")
.append("svg")
.attr("width", w)
.attr("height", h)
```


### Фигуры на основе данных:

Создаем данные для отрисовки фигур:
```ts
const dataset = [ 5, 10, 15, 20, 25 ];
```

Создаем круги:
```ts
circles = svg.selectAll("circle")
.data(dataset)
.enter()
.append("circle");
```

Стилизуем круги:
```ts
circles
.attr("cx", function(d, i) {
	return (i * 50) + 25;
})
.attr("cy", h/2)
.attr("r", function(d) {
	return d;
});
```

### Создание столбчатой диаграммы:

```ts
const w = 500;
const h = 100;

const svg = d3.select("body")
.append("svg")
.attr("width", w)
.attr("height", h)

svg.selectAll("rect")
.data(dataset)
.enter()
.append("rect")
.attr("x", function(d, i) {
	return i * (w / dataset.length)
})
.attr("y", 0)
.attr("width", 20)
.attr("height", 100)

```

**Можно добавить высоту в зависимости от данных:**
```ts
.attr("height", function(d) {
	return d * 4;
});
```

---
**Можно добавить выравнивание по высоте относительно нижней границы:**
```ts
.attr("y", function(d) {
	return h - d; // Height minus data value
})
.attr("height", function(d) {
	return d * 4; // Just the data value
});
```

**Результат:**

![[making_bar_chart_8 2.png|697]]

---
**Можно добавить цветовую заливку:**
```ts
.attr("fill", function(d) {
	return "rgb(0, 0, " + (d * 10) + ")";
});
```

---
**Можно добавить метки в виде текстовых данных:**
```ts
svg.selectAll("text")
.data(dataset)
.enter()
.append("text")
.text(function(d) {
	return d;
})
.attr("x", function(d, i) {
	return i * (w / dataset.length) + 5; // +5
})
.attr("y", function(d) {
	return h - (d * 4) + 15; // +15
})
.attr("font-family", "sans-serif")
.attr("font-size", "11px")
.attr("fill", "white");
```
