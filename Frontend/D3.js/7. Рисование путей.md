
Пути представляют более сложные объекты по сравнению с простыми примитивами. Путь позволяет отобразить набор любых примитивов с заполнением или без заполнения цветом. Для создания пути определяется элемент _path_:

```ts
const width = 400
const height = 400

const data = [
	{x: 80, y: 50},{x: 110, y: 80},{x: 140, y: 90},
	{x: 170, y: 70},{x: 200, y: 60},{x: 230, y: 60},
	{x: 260, y: 70},{x: 290, y: 80},{x: 320, y: 70}
];

// функция, создающая по массиву точек линии
const line = d3.svg.line()
	.x(function(d){return d.x;})
	.y(function(d){return d.y;});

const svg = d3.select("body").append("svg");

svg.attr("height", height).attr("width", width);

// добавляем путь
svg.append("path").attr("d", line(data));
```
