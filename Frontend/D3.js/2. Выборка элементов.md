
Выборка элементов является одной из первостепенных задач при работе с D3.js, так как чтобы создать графику, надо предварительно выбрать элемент, в котором эта графика будет размещаться.

Чтобы выбрать определенный элемент из структуры DOM, необходимо использовать один из следующих двух методов:

- `d3.select("selector")`: возвращает первый элемент из всех элементов, которые соответствуют селектору
- `d3.selectAll("selector")`: возвращает все элементы, которые соответствуют селектору

В качестве параметра в методы передается css-селектор элемента, например, id, название класса, название тега элемента, либо комбинация селекторов. Например:
```ts
const bodyElement = d3.select("body"); // выбираем элемент body
```

При этом мы можем использовать последовательно несколько уровней выборки:
```ts
d3.select("body").select("ul")
```
В данном случае сначала выбирается элемент body, потом из его элементов выбирается элемент ul.

---

### Операторы

Операторы представляют методы, которые проводят различные операции над выборкой. Существует несколько основных операторов:

- attr(): получает или устанавливает значение атрибута;
- style(): получает или устанавливает стиль элемента;
- text(): получает или устанавливает текстовое содержимое;
- html(): получает или устанавливает html-код элемента.

Если передать только один аргумент, то будет получено значение, если передать два аргумента то значение будет установлено в методах.

### Управление html-кодом элементов

Для управления html-элементfvb D3.js применяет ряд операторов, среди которых самыми распространенными являются следующие:

- append(): добавляет новый элемент, тег которого передается в метод в качестве параметра
- insert(): добавляет новый элемент по определенной позиции
- remove(): удаляет элемент

Добавим в элемент `body` какой-нибудь новый элемент:
```ts
d3.select("body").append("h2").text("Hello");
```

Метод `insert()` принимает два параметра:
```ts
d3.select("body").insert("h2", "ul").text("Hello Work");
```
Создаваемый элемент - в данном случае заголовок `<h2>` добавляется перед первым найденным в body элементом `<ul>`. Если же в элементе `body` нет элемента списка `<ul>`, то добавление идет в конец элемента `body` (аналогично применению метода `append()`)

Метод `remove()` удаляет элемент:
```ts
d3.select("h1").remove();
```
